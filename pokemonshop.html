Pokémon Card Shop

An Android app built in Unity that manages Pokémon card inventory using SQLite and C#.

Overview

This Unity-based application allows users to browse, update, amd purchase Pokémon cards. Designed for mobile Android devices, it combines a custom interface with persistent storage using SQLite.

Tech Stack

  • Unity with C# scripting
  • SQLite for embedded database
  • Designed for Android deployment

Features

  • Update cards - Name, Description, Price, Cost, Quantity
  • Visual card browser
  • Persistent storage using SQLite
  • Responsive UI optimized for mobile devices
