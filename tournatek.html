TournaTek

A co-founded SaaS platform that runs fully automated Rocket League tournaments — a Discord bot and web platform designed, built, and deployed single-handedly.

Visit TournaTek.com

The codebase is proprietary, so the source is not publicly available — the platform is live at the link above.

Overview

TournaTek is a live SaaS product that removes every ounce of manual admin work from running Rocket League tournaments. An organizer runs a single slash command in their Discord server, and the bot handles the entire lifecycle from there: player registration, check-in, team balancing, bracket generation, lobby credential delivery, match reporting, and results — all the way until a champion is crowned.

The platform is in production today, serving multiple Discord communities with registered players and completed tournaments. Alongside the bot, a companion website provides public player profiles, career statistics, and server leaderboards, with rank data pulled from a Rocket League stats API.

I co-founded TournaTek and am its sole developer — every line of the bot, database, and website is my work. This is my most ambitious project and the one with the most engineering behind it.

Tech Stack

  • Python
  • discord.py
  • Flask
  • PostgreSQL (Neon)
  • Rocket League Stats API (integration)
  • Discord API
  • Render (cloud hosting)

Features

  • Dual-mode AI team balancing — exhaustively evaluates every roster combination for small lobbies, and switches to an adaptive optimization algorithm when combinations run into the millions
  • Four bracket formats: single elimination, double elimination, round robin, and Swiss
  • Rank gates that restrict tournaments to specific competitive tiers
  • Crash-resilient by design — tournament state is persisted in PostgreSQL, so the bot survives restarts mid-tournament without losing data
  • Fully native Discord experience: live bracket embeds, one-click match reporting, instant lobby credentials, and self-cleaning channels
  • Career statistics, weighted MMR, and server leaderboards published to the web platform

My Role

  • Co-founder — shaped the product vision, feature set, and roadmap
  • Sole developer of the entire stack: the discord.py bot, the PostgreSQL schema on Neon, the Flask web application, and the Rocket League stats API integration
  • Designed the tournament engine, including the team-balancing algorithms and crash-recovery model
  • Own deployment and operations end-to-end, hosting the platform on Render for real users in production
