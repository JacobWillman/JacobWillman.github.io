Cars2U

A full desktop app for vehicle ordering and management built in Python, SQLite, and Tkinter.

GitHub Installer

Overview

This application allows customers to browse vehicles, add to cart, and checkout with a simulated payment system. It also features an employee dashboard, inventory management, and real-time discount logic.

Tech Stack

  • Python (Tkinter for GUI)
  • SQLite for database storage
  • HTML/CSS for generated reports

Features

  • Login/registration system
  • Cart and checkout with promo codes
  • HTML report generation
  • Favorites system
  • Employee inventory management
