Ant Simulation

A multithreaded Java simulation where a colony of ants searches for food and lays down pheromone trails to find the shortest paths home.

GitHub

Overview

This simulation models a colony of ants that leave their nest, search for food, and lay down pheromone trails as they travel back and forth. As stronger trails build up over time, the ants gradually shift from wandering randomly to following the best paths, so the colony ends up finding efficient routes to food entirely on its own.

Tech Stack

  • Java
  • Multithreaded simulation engine
  • Object-oriented entity/class design

Features

  • Pheromone-trail pathfinding that emerges from ant behavior, not hardcoded routes
  • Worker and Scout ant types with distinct movement and trail-following logic
  • Simulation logic and rendering run on separate threads for smooth performance
  • Thread pool updates hundreds of ants concurrently
  • Shared world state is locked during updates/draws to keep threads in sync
