Space Attack

A fast-paced Unity Android game where players blast asteroids, collect powerups, and compete for high scores stored in SQLite.

Overview

Space Attack is an arcade-style 2D shooter developed in Unity for Android devices. Players control a spaceship, shoot randomly generated asteroids of varying health, and collect helpful items like shields, speed boosts, and health packs. A high score system using SQLite keeps track of the playerâ€™s top runs.

Tech Stack

  • Unity (C# scripting)
  • SQLite for local high score tracking
  • Targeted for Android platform

Features

  • 2D space shooter gameplay with responsive controls
  • Randomized asteroid spawns and progressive difficulty
  • Pickups - Shield, Bomb, Double gun, Tripple gun, Lose a Life
  • Persistent local high scores via SQLite
  • Mobile-optimized user interface
